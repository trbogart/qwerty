# qwerty
Calculate words based on finger patterns with QWERTY or other keyboard layouts.

Keyboard layout can be set by editing `common.py`. 
- TODO: use arg or calculate for all layouts

## Programs
### `qwerty8.py`

Words that use each finger exactly once with a keyboard layout.

### `qwerty8_2.py`

Pairs of 2 words that use each finger exactly once with a keyboard layout.

### `qwerty2_multi.py`

All groups of words that use each finger exactly once with a keyboard layout.

### `qwerty1.py`

Words of various lengths that use a single finger with a keyboard layout.
